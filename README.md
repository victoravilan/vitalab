Need help or want to join the community? Join our [Discord](https://discord.gg/shDEGBSe2d).

To run the devserver:
```
npm install
npm run dev
```

## PWA Configuration

This application is now configured as a Progressive Web App (PWA). This includes:

*   A web app manifest (`public/manifest.json`) that defines the application's metadata.
*   Icons for different devices, located in the `public/img` folder.
*   The application is now installable on mobile devices.
